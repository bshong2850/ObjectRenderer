Last Version 2020 01 17

OBJ, STL, BV 파일 Rendering 프로젝트 입니다.

OBJ
 - vertex와 face가 있는 파일만 읽을 수 있습니다(2020.01.17 현재)

STL
 - Ascii 및 Binary File 모두 읽을 수 있습니다.

BV
 - OpenMesh 프로젝트에서 Boundray만 추출하면 나오는 vertex 좌표 모음입니다.
 - vertex position에 Point를 생성합니다.

사용법
 - main문에 원하는 파일명을 입력하거나 x64/Release/에 있는 exe 파일에 원하는 object 파일을 끌어다 넣습니다.

기능
 - 마우스 좌 클릭한 상태로 움직임 - Rotation
 - 마우스 가운데 클릭한 상태로 움직임 - Translate
 - 마우스 우 클릭한 상태로 위아래로 움직임 - Zoom
